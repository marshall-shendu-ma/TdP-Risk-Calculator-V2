TdP Risk Assessment Categorization (TRAC) Calculator

Introduction

The TdP risk assessment categorization calculator is used to assess the TdP risk of small molecule compounds using three (3) hiPSC-CM electrophysiological response predictors as part of the evolving Comprehensive In Vitro Proarrhythmia Assay (CiPA) paradigm.

Predictor Inputs

Did drug-induced arrhythmias occur at any concentration? 0=no arrhythmia, 1=type A arrhythmia, 2=any other arrhythmia type.

Maximum repolarization change observed at any concentration (ms). (ΔΔFPDc or ΔΔAPD90c)

Drug-induced repolarization change at expected Cmax (ms). (ΔΔFPDc or ΔΔAPD90c)

Program Outputs

Model 1 TdP Risk

Instructions

  1. For details on how to use the hiPSC-CM CiPA TdP Risk Assessment Categorization (TRAC) Calculator, read the Context of Use (CoU), which includes limitations of use.
  2. Insert the model predictor input parameters.
  3. Click Calculate
  4. Read the outputs in the “Program outputs” section to obtain the TdP risk categorization estimated by the set of input parameters.
  5. Graphical depiction of the estimated TdP risk potential will be displayed for Model 1 TdP Risk (Click Switch Model to display Model 2 TdP Risk results).
  6. To use the Cmax interpolation feature add the input parameters and click Calculate.
