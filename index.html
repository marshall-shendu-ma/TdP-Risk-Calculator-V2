Categorization of TdP risk dependent on compound electrophysiological response predictors

TdP Risk Categorization Calculator Program

Instructions

  1. For details on how to use the hiPSC-CM CiPA TdP Risk Categorization Tool, read the Context of Use (CoU), which includes limitations of use.
  2. Insert the model predictor input parameters in the “TdP input parameters” table. Hover over the information icons next to each section header for parameters definition and range of use.
  3. Read the outputs in the “Program outputs” table to obtain the TdP risk categorization estimated by the set of input parameter selected.
  4. Graphical depiction of the estimated TdP risk potential will be displayed in “TdP risk categorization estimation” section. The concentration-response curves (i.e., Hill Curve) will change accordingly showing the estimated concentration maximum (Cmax) for the set of pulse parameter selected.
