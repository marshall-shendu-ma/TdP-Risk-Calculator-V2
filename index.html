TdP Risk Categorization Calculator Program

Instructions

  1. For details on how to use the hiPSC-CM CiPA TdP Risk Categorization Tool, read the Context of Use (CoU), which includes limitations of use.
  2. Insert the model predictor input parameters in the “Cmax Extrapolation” section below (click the triangle to expand/collapse).
  3. Alternatively, supply Predictor 1, Predictor 4, and Predictor 7 values manually in the “Predictor Inputs” section.
  4. Read the outputs in the “Program Outputs” section to obtain the TdP risk categorization estimated by the set of input parameters selected.
  5. Graphical depiction of the estimated TdP risk potential will be displayed in “Program Outputs.”
